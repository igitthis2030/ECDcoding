# PayFast Demo — ECDcoding\n\nThis repository contains a small Express demo server to exercise PayFast sandbox payments and subscriptions.\n\nQuick start\n\n1. Copy the example env and edit values (do NOT commit real credentials):\n\n   cp .env.example .env\n\n2. Ensure you are running Node.js 18+ (the demo uses the native fetch API).\n\n3. Install dependencies and start:\n\n   npm install\n   npm start\n\n4. Expose port 3000 so PayFast can reach /ipn (ngrok or Codespaces public port). Set PUBLIC_BASE_URL in .env to the public URL and restart.\n\nNotes & security\n\n- .env is ignored; keep real credentials in GitHub Secrets and Codespaces secrets.\n- This demo uses in-memory stores for orders and subscriptions — not for production.
